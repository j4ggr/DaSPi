    Estimator

    daspi.statistics.estimation.Estimator

    agreement: int | float property writable

    Get the multiplier of the sigma agreement for Cp and Cpk value (default 6). By setting this value the cp and cpk values are resetted to None.

    If setting agreement by giving the percentile, enter the acceptable proportion for the spread, e.g. 0.9973 (which corresponds to ~ 6 sigma)

    dist: rv_continuous property

    Get fitted distribution. None if method distribution has not been called.

    dist_params: Tuple[float, ...] property

    Get params of fitted distribution. None if method distribution has not been called.

    dof: int property

    Get degree of freedom for filtered samples

    excess: float property

    Get the Fisher kurtosis (excess) of filtered samples. Calculations are corrected for statistical bias. The curvature of the distribution corresponds to the curvature of a normal distribution when the excess is close to zero. Distributions with negative excess kurtosis are said to be platykurtic, this distribution produces fewer and/or less extreme outliers than the normal distribution (e.g. the uniform distribution has no outliers). Distributions with a positive excess kurtosis are said to be leptokurtic (e.g. the Laplace distribution, which has tails that asymptotically approach zero more slowly than a Gaussian, and therefore produces more outliers than the normal distribution): - excess < 0: less extreme outliers than normal distribution - excess > 0: more extreme outliers than normal distribution

    filtered: pd.Series property

    Get the samples without error values and no missing value

    lcl: float property

    Get lower control limit according to given strategy and agreement.

    mean: float property

    Get mean of filtered samples

    median: float property

    Get median of filtered samples

    n_missing: int property

    Get amount of missing values

    n_samples: int property

    Get sample size of unfiltered samples

    p_ad: float property

    Get the probability that the filtered samples are subject of the normal distribution by performing a Anderson-Darling test.

    p_dist: float property

    Get probability of fitted distribution. None if method distribution has not been called.

    p_excess: float property

    Get the probability that the excess of the population that the sample was drawn from is the same as that of a corresponding normal distribution.

    p_skew: float property

    Get the probability that the skewness of the population that the sample was drawn from is the same as that of a corresponding normal distribution

    q_low: float property

    Get quantil for lower control limit according to given agreement. If the samples is subject to normal distribution and the agreement is given as 6, this value corresponds to the 0.135 % quantile (6 sigma ~ 99.73 % of the samples).

    q_upp: float property

    Get quantil for upper control limit according to given agreement. If the sample data is subject to normal distribution and the agreement is given as 6, this value corresponds to the Q_0.99865 (0.99865-quantile or 99.865-percentile).

    samples: pd.Series property

    Get the raw samples as it was given during instantiation as pandas Series.

    sem: float property

    Get standard error mean of filtered samples

    skew: float property

    Get the skewness of the filtered samples. Calculations are corrected for statistical bias. For normally distributed data, the skewness should be about zero. For unimodal continuous distributions, a skewness value greater than zero means that there is more weight in the right tail of the distribution: - skew < 0: left-skewed -> long tail left - skew > 0: right-skewed -> long tail right

    std: float property

    Get standard deviation of filtered samples

    strategy: Literal['eval', 'fit', 'norm', 'data'] property writable

    Strategy used to determine the control limits (can also be interpreted as the process range).

    Set strategy as one of {'eval', 'fit', 'norm', 'data'} - eval: If no evaluate function is given, the strategy is determined according to the internal evaluate method. - fit: First, the distribution is searched for that best represents the process data and then the process variation tolerance is calculated - norm: it is assumed that the data is subject to normal distribution. The variation tolerance is then calculated as agreement * standard deviation - data: The quantiles for the process variation tolerance are read directly from the samples.

    ucl: float property

    Get upper control limit according to given strategy and agreement.

    __init__(samples, strategy='norm', agreement=6, possible_dists=DIST.COMMON, evaluate=None)

    An object for various statistical estimators

    The attributes are calculated lazily. After the class is instantiated, all attributes are set to None. As soon as an attribute (actually Property) is called, the value is calculated and stored so that the calculation is only performed once

    Parameters:

    Name Type Description Default
    samples NumericSample1D

    sample data

    		 required
    strategy (eval, fit, norm, data)

    Which strategy should be used to determine the control limits (process spread): - eval: The strategy is determined according to the given evaluate function. If none is given, the internal evaluate method is used. - fit: First, the distribution that best represents the process data is searched for and then the agreed process spread is calculated - norm: it is assumed that the data is subject to normal distribution. The variation tolerance is then calculated as agreement * standard deviation - data: The quantiles for the process variation tolerance are read directly from the data. by default 'norm'

    		 'eval'
    agreement int or float

    Specify the tolerated process variation for which the control limits are to be calculated. - If int, the spread is determined using the normal distribution agreementsigma, e.g. agreement = 6 -> 6sigma ~ covers 99.75 % of the data. The upper and lower permissible quantiles are then calculated from this. - If float, the value must be between 0 and 1.This value is then interpreted as the acceptable proportion for the spread, e.g. 0.9973 (which corresponds to ~ 6 sigma) by default 6

    		 6
    possible_dists tuple of strings or rv_continous

    Distributions to which the data may be subject. Only continuous distributions of scipy.stats are allowed, by default DIST.COMMON

    		 COMMON
    evaluate callable or None

    Function that takes this instance as argument and returns one of the allowed strategy {'eval', 'fit', 'norm', 'data'}, by default None

    distribution()

    Estimate the distribution by selecting the one from the provided distributions that best reflects the filtered data.

    Returns:

    Name Type Description
    dist scipy.stats rv_continuous

    A generic continous distribution class of best fit
    p float

    The two-tailed p-value for the best fit
    params Tuple[float, ...]

    Estimates for any shape parameters (if applicable), followed by those for location and scale. For most random variables, shape statistics will be returned, but there are exceptions (e.g. norm). Can be used to generate values with the help of returned dist
    Notes

    First, the p-score is calculated by performing a Kolmogorov-Smirnov test to determine how well each distribution fits the samples. Whatever has the highest P-score is considered the most accurate. This is because a higher p-score means the hypothesis is closest to reality.

    evaluate()

    Evaluate strategy to calculate control limits. If no evaluate function is given the strategy is evaluated as follows:

    1. If variance is not stable within the samples -> strategy = 'data'
    2. If variance and mean is stable and samples follow a normal curve -> strategy = 'norm'
    3. If variance and mean is stable but samples don't follow a normal curve -> strategy = 'fit'
    4. If variance is stable but mean not and samples follow a normal curve -> strategy = 'norm'
    5. If variance is stable but mean not and samples don't follow a normal curve -> strategy = 'data'

    Returns:

    Name Type Description
    strategy {fit, norm, data}

    Evaluated strategy to calculate control limits
