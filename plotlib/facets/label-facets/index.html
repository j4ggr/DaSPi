    Label facets

    daspi.plotlib.facets.LabelFacets

    A class for adding labels and titles to facets of a figure.

    Parameters:

    Name Type Description Default
    figure Figure

    The figure to label.

    		 required
    axes 2DArray

    A 2D array of Axes instances. Setting squeeze=False when using the plt.subplots method ensures that it is always a 2D array.

    		 required
    fig_title str

    Main title that should be displayed at the top of the chart, by default ''.

    		 ''
    sub_title str

    Subtitle, which should appear directly below the main title and slightly smaller than it, by default ''.

    		 ''
    xlabel str | Tuple[str, ...]

    The axis label(s) of the figure. To label multiple axes with different names, provide a tuple; otherwise, provide a string, by default ''.

    		 ''
    ylabel str | Tuple[str, ...]

    The axis label(s) of the figure. To label multiple axes with different names, provide a tuple; otherwise, provide a string, by default ''.

    		 ''
    info bool or str

    Indicates whether to include an info text at the lower left corner in the figure. The date and user are automatically added, by default False.

    		 False
    rows Tuple[str, ...]

    The row labels of the figure, by default ().

    		 ()
    cols Tuple[str, ...]

    The column labels of the figure, by default ().

    		 ()
    row_title str

    The title of the rows, by default ''.

    		 ''
    col_title str

    The title of the columns, by default ''.

    		 ''
    axes_titles Tuple[str, ...]

    Title for each Axes, usefull for JointCharts, by default ()

    		 ()
    legend_data Dict[str, LegendHandlesLabels]

    The legends to be added to the figure. The key is used as the legend title, and the values must be a tuple of tuples, where the inner tuple contains a handle as a Patch or Line2D artist and a label as a string, by default {}.

    		 {}

    axes: NDArray = axes instance-attribute

    A 2D array containing the Axes instances of the figure.

    axes_titles: Tuple[str, ...] = axes_titles instance-attribute

    The titles of each axes.

    col_title: str = col_title instance-attribute

    The title of the columns.

    cols: Tuple[str, ...] = cols instance-attribute

    The column labels of the figure.

    fig_title: str = fig_title instance-attribute

    The title to display at the top of the chart.

    figure: Figure = figure instance-attribute

    The figure instance to label.

    info: bool | str = info instance-attribute

    Indicates whether to include an info text in the figure.

    legend: Legend | None property

    Get legend added to figure (read-only).

    legend_artists: List[Artist] property

    Get legend artists (read-only).

    legend_data: Dict[str, LegendHandlesLabels] = legend_data instance-attribute

    The legend_data to be added to the figure.

    row_title: str = row_title instance-attribute

    The title of the rows.

    rows: Tuple[str, ...] = rows instance-attribute

    The row labels of the figure.

    shift_fig_title: float property

    Get offset in y direction for fig title (read-only).

    shift_legend: float property

    Get offset in x direction for legend (read-only).

    shift_sub_title: float property

    Get offset in y direction for sub title (read-only).

    shift_text_x: float property

    Get offset to move text based on the fig width (read-only).

    shift_text_y: float property

    Get offset to move text based on the fig height (read-only).

    sub_title: str = sub_title instance-attribute

    The subtitle to display directly below the title of the chart.

    xlabel: str | Tuple[str, ...] = xlabel instance-attribute

    The x-axis label(s) of the figure.

    ylabel: str | Tuple[str, ...] = ylabel instance-attribute

    The y-axis label(s) of the figure.

    add_axes_titles()

    Add the provided axes titles.

    add_col_labels()

    Add column labels and column title to the figure.

    add_info()

    Insert an info text in the bottom left-hand corner of the figure. By default, the info text contains today's date and the user name. If attribute info is a string, it is added to the info text separated by a comma.

    add_legend(handles, labels, title)

    Adds a legend at the right side of the figure. If there is already one, the existing one is extended with the new one

    Parameters:

    Name Type Description Default
    handles Tuple[Patch | Line2D, ...]

    A list of Artists (lines, patches) to be added to the legend.

    		 required
    labels Tuple[str, ...]

    The labels must be in the same order as the corresponding plots were drawn. If no labels are given, the handles and labels of the first axes are used.

    		 required
    title str

    Title for the given handles and labels.

    		 required

    add_row_labels()

    Add row labels and row title to the figure.

    add_titles()

    Add the figure and sub-title at the top of the chart.

    add_xlabel()

    Add x-axis label(s) to the figure.

    add_ylabel()

    Add y-axis label(s) to the figure.

    draw()

    Draw all the label facets to the figure.

    get_legend_artists(legend) staticmethod

    Get the inner children of a legend.

    Parameters:

    Name Type Description Default
    legend Legend

    The legend object.

    		 required

    Returns:

    Type Description
    List[Artist]:

    The artists representing the inner children of the legend.
